# Prime Inventory v0.4

Telegram WebApp для инвентаризации бара.

## Что умеет

- импорт файла остатков XLSX/XLS/CSV;
- автоматическое создание складов из строк `Склад: ...`;
- перевод `л` в `мл`, `кг` в `г`, `шт` в `шт`;
- отрицательные остатки из iiko делает положительными;
- автоматическое создание базы товаров из файла;
- инвентаризация по складам: Бар, Бэк, Холодильник, Морозилка, Склад;
- режимы ввода по типу товара;
- крепкий алкоголь: вес или мл, без режима шт;
- быстрый ввод 25% / 50% / 75% / Full;
- экран отклонений в реальном времени;
- экран непосчитанных товаров;
- экспорт CSV;
- отправка результата в Telegram через `tg.sendData()`.

## Запуск локально

```bash
npm install
npm run dev
```

## GitHub Pages

Проект уже настроен под GitHub Pages через `vite.config.js`:

```js
base: './'
```

Для публикации:

```bash
npm run build
```

Загрузи содержимое папки `dist` в GitHub Pages.

Либо подключи GitHub Actions для автоматического деплоя.
